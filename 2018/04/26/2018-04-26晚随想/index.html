大半夜思考 人生啊呀呀

发表于 2018-04-26 | 分类于 美好的人 | | 阅读次数:

最近可能是感冒了，也可能是最近有点累，总是想一些乱七八糟的东西。北漂一年多了，生活貌似依旧那样，虽然拿着一份还不错的薪水，可是漂了这么久，自己依旧没有一点储蓄，想来也可笑。总是用以后可以赚更多钱为理由借口，而对储蓄理财一点也不去关心。那天和朋友聊天，聊起理财，才稍有打算，愿为时不晚。
加班已经成为一种常态，放弃了稳定安逸，看起来前途很光明的工作，跳槽进了一家非常初创的企业，当时做这个决定，非常坚决，觉得年轻，就应该去累点，忙点，嗯！这貌似并没有什么错，是需要累点，可是总有数不出的苦和累，自己总是不知足，闲的时候羡慕忙的生活，忙的时候，总想着过得简单点。当初是抱着，自己能够加入创业公司，然后疯狂成长，公司IPO，财富自由，人生完美。即使失败，也会感谢这段疯狂的经历，这是最快的成长方式，总说别人想的多，自己貌似想的也多，然后依旧会是错的。
不知道要说点什么，也不知道要写给谁去看，想找个人好好去聊聊天，能够聊个几百的，可是想想周围的人，能够去聊的人，又寥寥无几，我好希望有一个高人，来解答我心中的疑惑，想和他们聊个通宵，聊个不眠不休。唯一能想起的是朋哥，也不知道五一他是否忙，是否会去福州，看他的小孩和嫂子。
小时候所认为的公平，法律 正义，现在来看，统统被现实打败，那是只是部分人所拥有的，正如朋哥之前所言，社会诸如金字塔，少数人在上面，大多数人在底层，然而少数人总是掌握着大部分资源，从下往上的通道越来越少，而金字塔越累越高。曾以为我们所生活的时代，是一个亘古未有的和平，稳定，公平，充满着正义的时代，可是最近发现，吊打了自己的人生观，现在依旧和过去差不了多少，那时候的人，也应该觉得封建社会，皇权，大人是至上的，大家都认同的，都理所当然的遵从，就像现在我觉得的，法律，法规，也是至上的，那是公平的。细说，经不起细究，可能自己所谓的方法论还没养成，也正处在一个转变的过程中，痛并快乐的成长着。
自己所曾拥有的那一份感情，也支离破碎，有时候反省自己，那段感情可能自己也就是别人口中那种渣男，不负责任的人。纵去然也，感谢人生所遇的一路路，一个个的人们，感觉有你们，请多多包涵对你们不经意的刮伤，我，只愿世界唯好。
茫茫人生路，有时候在睡觉的哪一瞬间，想着自己为什么而活着，人又为什么存在，生生死死究竟要造就什么。是不是死了，真会有天堂，会不会有来世。活着的人，为什么又会有哪些贫富差距，也许现在所处的时代，正如盛唐时代，朱门酒肉臭路有冻死骨的。然而，依旧 世界依旧转的正正规规，我只不过随波逐流，被浪所拍打，被水所浸湿。可悲的事，庸庸碌碌，找不到一丝转变的机会，那天给国建说，他的这段时间，是他人生的最低谷，其实我，又何尝不是呢！这段正在经历着的刻苦铭心的北漂，孤独，无助，疲惫，疯狂，夜不能寐，似一场修行，也似一场赌注，压青春为注，博功成名就。有时真怕，那一瞬间感觉就要倒下，也怕有那么一瞬，自己坚持不下去了，放弃了，走不下去了，那将会又是什么呢？
不知自己所思所想，是不是因为自己看了朋友圈的一段抖音，就变得，总想做点什么。想去表现点什么，也或许是自己寂寞了。自己真的好好笑啊！不知道若干天后的自己看到此文，会是什么呢？看一个小视频，竟然觉得是所谓的一见钟情哈！看来自己还真是小呢！一点也不成熟，毛毛糙糙，还想高中的时候那样，去焦灼的等待的微信，等待着QQ.等待那一丝丝点点动态。偶不知该如何，又不知如何洗去这段不该有的想法，控制不住自己，就像没有毅力坚持早睡早起一样。就忽然觉得，有个人会是那样的完美，看到身边美好的事物，总会联想到那个人，可能那个人终究是不存在，可能那个人也存在。
最近也发现，其实扪心自问，自我反省，感觉自己就像生活在那种很自我世界里，而无法自拔。还是活在那个所谓的中二年代，耿直、不变通而且不够世故、圆滑、聪明的时代。看过一部电影 《暴雪将至》 也许我就如其中那个男主人公一样，生活在那样一个自我世界里。自己所为人的世界，是自己会有能力去实现一些东西，是自己有可能去改变一些东西，是什么，貌似也说不清楚，总是相信未来会更好，凭借自己的努力，可以得到想要的，好的生活，被人的羡慕那中生活么？不知道。认为自己可以去做一番大事业，喜欢历史，喜欢那些攻城略地，成就大业的千古英雄。觉得商业诸如战争诸如乱世，也会造就英雄，喜欢那种凭借自己无限的想法，做出别人无法做出的事情，获得别人赞赏，获得别人认可，让自己🉐有面子，假若天之垂怜，实现财富自由，过上 所谓有钱没有资源那种 多维富豪的生活。然而，内心深处是不喜欢这样的。内心深处，想要是什么？有很多，但是无法言表出来，
想了好久，也累了 该睡觉了，最后也发现，这是一个阶段，是一个迎来改变的阶段，可能每个人一生中都会有那么几个转折的点，那么几个睡不着的夜晚，就看怎么去把握，怎么去转变了，想到的解决办法，就是在多读几本书，复盘下自己，那块不足，就努力补起来，想办法，别总是认为学好技术就好了，成为技术牛人，但是为人处世却单单纯纯，像个小孩子的思维，还是不行的，慢下步子，边奋斗边生活。
最后还是觉得，朋友圈那小姑娘好可爱，真是触及了我的内心，哇塞，忍不住多看几眼，好像和她聊聊啊！😄竟然好怕自己，这种直男的性格，吓跑人家。嗯，明天找机会和她聊聊啦！美好感觉 。
晚安
