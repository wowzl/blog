Office Web Apps Server 是一种 Office 服务器产品，可提供在浏览器中查看和编辑 Office 文件的服务。可提供浏览器版 Word、PowerPoint、Excel 和 OneNote。

环境

1.阿里云服务器 1核2G 服务器两台
2.同属于统一网关

所需软件

  1. Windows Server 2012 R2
  2. .NET Framework 4.5.2
  3. Office Web Apps Server cn_office_web_apps_server_2013_with_sp1_x64_dvd_3832995.iso
  4. Office Web Apps Server SP1 wacserversp2013-kb2880558-fullfile-x64-glb.exe
  5. wacserverlanguagepack.exe
